記錄一下各種好用的 tool

  • Ruby on Rails 裝機趴 (Only Mac)

Mac Tool

  • PostMan - API test
  • OmniDiskSweeper - 清除檔案
  • f.lux - 眼睛保護
  • Owly - 停止螢幕關閉
  • Gas Mask - host 管理
  • Alfrd - 快速搜尋
  • Macs Fans Control - mac 風扇控制
  • Sip - color
  • MacDown - markdown editor
  • Sequel Pro - mysql
  • Postico - postgresql
  • SqliteBrower - sqlite
  • Welly - PTT
  • Magnet - 視窗分割
  • AppCleaner - 深度清除 APP
  • Trailer.app - Accelerate your GitHub workflow. Never miss a comment again.
  • getbitbar - Put anything in your Mac OS X menu bar

Terminal Tool

  • the_silver_searcher - 快速搜尋檔案，比 grep 還快
  • htop - 比 top 好用的監控工具
  • tig - Text-mode interface for git
  • wrk - http benchmark
  • Ngrok - 內網分享
  • z - jump around
  • autojump - A cd command that learns - easily navigate directories from the command line (跟 z 類似)
  • peco - 可以將一些指令變成互動式的指令 e.g. git checkout

UI Framework

  • normalize.css - 將一般常用的 css 先處理好
  • element UI - for vue.js
  • vuetify - for vue.js
  • material-ui
  • materializecss
  • ant.design
  • bootstra
  • bulma
  • uikit
  • semantic-ui

  • milligram

Code Quality

  • flay - 找出相似的程式碼讓 code 更 dry
  • rubocop - codeing style
  • simplecov - 測試覆蓋率
  • brakeman - 檢查安全性
  • Bullet - 檢查 N+1 Query
  • rubycritic - 靜態分析程式碼(Reek, Flay 和 Flog)
  • pronto - review 完後 commit 在 github comment 上面
  • traceroute - 找出沒有在使用的 router
  • rack-mini-profiler - profiling 每一頁花的時間
  • rails_panel + RailsPanel Chrome - 在chrome的developer tool加上rails的分析工具。
  • rails_best_practices - 檢查rails專案中是否有違反 rails best practices。
