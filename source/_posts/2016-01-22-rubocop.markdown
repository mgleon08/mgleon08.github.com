rubocop 像是一個程式評量工具，會告知在 rails 中的寫法，要如何才會比較好!

<!-- more -->

```ruby
gem install rubocop
```

###指令

```ruby
rubocop app spec lib/something.rb
```

###自訂檔案
.rubocop.yml

```ruby
inherit_from:
  - .ruby-style.yml
```
.ruby-style.yml

```ruby
AllCops:
  Include:
    - '**/Rakefile'
    - '**/config.ru'
  Exclude:
    - 'db/**/*'
    - 'config/**/*'
    - 'script/**/*'
```
