若專案依賴各種外部的進程使用 foreman 來管理它們。

<!-- more -->

#設定
在根目錄底下新增 Procfile 檔案

設定所有要跑的指令

```ruby
#Procfile
web:    bundle exec rails s -p $PORTworker: bundle exec rake workerurgentworker: bundle exec rake urgent_workerscheduler:  bundle exec rake scheduler
```

#安裝gem foreman

```ruby
gem install foreman
```
