使用 Octopress + Github Pages 寫 Blog

Dec 11th, 2015 11:15 pm | Comments

Install Octopress

先到 Octopress 將檔案 git clone下來

開新的 Github Repository

到 Github 點選 New repository, Repository name 輸入自己帳號的名稱

像筆者帳號 mgleon08 就輸入 mgleon08.github.com

佈署到Github上

rake generate 產生靜態檔案
rake deploy 部署到github

Push到Github

在本機觀看

接著打開localhost:4000就可以看到了!!

發表文章

Octopress會幫你產生新的檔案 2015-12-08-title.markdown 到source/_posts/下面，有發現後面是用markdown嗎?

沒錯，Octopress 主要是用 Markdown 語法來撰寫，筆者是使用 MacDown這套來做編輯。

對Markdown語法還不了解的可以到Markdown教學看看。

撰寫好後先rake preview預覽一下畫面
確認好後就在rake deploy，這樣網路上的就會更新了。
