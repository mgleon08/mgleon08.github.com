underlying type

  • time/Duration

在 time package 定義了一個 type Duration 底層的 type 是 int64

但是當 Duration 和 a 相乘會發生 invalid operation: h *= a (mismatched types time.Duration and int64) 因為雖然底層都是 int64 但 golang 是強型別，不允許不同的 type 作處理，因此必須再透過 time.Duration(a) 轉成 Duration type

只要底層一樣就可以做轉換 time.Duration(a) ounce(g)
